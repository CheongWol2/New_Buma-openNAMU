## New Buma
 * 오픈나무 Buma를 최대한 기존 스킨처럼 변경한 스킨입니다.

## 개발 상황
 * 95% 검색 버그 수정 

## 적용 방법
 * 다운 받고 폴더 이름을 "newbuma"로 변경해서 "views"에 넣어주세요.

## 개발 예정
 * 클릭 시 탭 변경 (노가다가 될 것으로 예상)

## 개발 불가
 * 구조 파악 완료. 모두 구현 가능함.

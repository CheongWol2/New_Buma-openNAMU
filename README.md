## New_Buma-openNAMU
오픈나무 Buma를 최대한 기존 스킨처럼 변경한 스킨입니다.

## 적용 방법
다운 받고 폴더 이름을 "newbuma"로 변경해서 "views"에 넣어주세요.

현재 스킨 개조 중
